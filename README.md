# Chennai Super Kings Fan Website

## Objective

To design and develop a responsive fan website dedicated to the Chennai Super Kings (CSK) cricket team, showcasing player information, team statistics, match schedules, and latest news updates in an engaging and user-friendly interface.

## Features

* **Hero Section:** Eye-catching welcome banner with CSK-themed design.
* **Navigation Menu:** Easy access to Home, Players, Matches, Statistics, News, and Gallery sections.
* **Star Players Showcase:** Interactive player cards displaying key CSK players.
* **Team Statistics:** Highlights important team achievements and performance metrics.
* **Upcoming Matches:** Organized schedule table displaying future fixtures.
* **Latest News Section:** Displays recent team updates and announcements.
* **Responsive Design:** Optimized for desktop, tablet, and mobile devices.
